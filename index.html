¿Qué es Marcador?

Un marcador es una característica del navegador web que se utiliza para guardar la dirección URL de un sitio web para referencia futura . Los marcadores ahorran tiempo al usuario y al navegador, lo que es especialmente útil para páginas web con direcciones URL largas o para acceder a una parte específica del sitio que podría no ser la página de inicio del sitio.
